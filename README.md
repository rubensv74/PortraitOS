# PortraitOS

> Professional Portrait Intelligence Platform

PortraitOS es una aplicación local diseñada para construir retratos profesionales mediante IA preservando rigurosamente la identidad real de la persona.

No es un generador de prompts.

Es un sistema de dirección creativa.

---

# Filosofía

PortraitOS parte de un principio innegociable:

> La identidad nunca se modifica.

La creatividad únicamente puede aplicarse sobre:

- iluminación
- composición
- pose
- vestuario
- escenario
- cámara
- color
- profundidad

Nunca sobre:

- edad
- cabello
- canas
- mandíbula
- nariz
- ojos
- labios
- piel
- arrugas
- asimetrías

# Principio fundamental

Toda decisión del sistema debe responder a una única pregunta:

> ¿Esto mejora el retrato sin alterar la identidad?

Si la respuesta es NO...

...la acción queda prohibida.
